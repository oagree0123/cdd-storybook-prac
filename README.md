# otory-book
> storybook을 활용한 컴포넌트 주도 개발 연습

## 개발환경
- pnpm
- React
- Styled-componenets

## Installation

```sh
$ npm i otory-book
```
```sh
$ pnpm add otory-book
```
```sh
$ yarn add otory-book
```

## Usage

### Serving the app

```jsx
import { Button } from "./components/Button/Button";

const App = () => {
  return (
    <Button primary='primary' size='medium'>Button</Button>
  )
}
```
